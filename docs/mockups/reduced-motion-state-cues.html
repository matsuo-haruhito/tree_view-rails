State cue comparison without animation

Loading in progress

Use a stable pending row, busy attribute, explicit status label, and disabled action instead of depending on a spinner.

tree name state cue action
loadpending Loading folder loading children
  • Stable text keeps the state readable if spinner animation is removed.
  • aria-busy="true" and the disabled action describe the same pending boundary.

Retry or error state

Combine the retry affordance with an error row shape and plain status text so the failed state is not color-only.

tree name state cue action
retryerror Failed folder load failed
  • Retry copy appears in both the toggle control and the action column.
  • The row background, status pill, and explicit label carry the cue without motion.

Current and selected state

Keep the current row visible through the row state, aria-current, a leading edge, and a selected label.

  • Final navigation and route selection remain host-app responsibilities.
