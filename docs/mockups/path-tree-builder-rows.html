Responsibility boundary

Generated folder row
TreeView provides the hierarchy row, folder marker, toggle affordance, depth cue, and child count.
Record row
The host app owns visible record columns, status labels, and business actions attached to the terminal path item.
Outside this mock
Routes, downloads, authorization copy, and final file-manager behavior stay outside the gem mockup.
