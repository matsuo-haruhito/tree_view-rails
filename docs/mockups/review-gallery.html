tree_view-rails

TreeView mockup review gallery

Static review hub for comparing the current TreeView mockup set without running a Rails app. Each card keeps the linked mockup as the source of truth and adds enough framing to compare structure, state cues, and responsibility boundaries in one session.

Back to mockup README Open docs index

How to use this gallery

  • Start here when a review needs quick side-by-side comparison across the baseline and focused mockup surfaces.
  • Open the linked full mockup when you want the complete page context or longer notes for one state family.
  • Treat final business copy, routes, permissions, and actions as host-app responsibilities.
