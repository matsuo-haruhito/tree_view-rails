tree_view-rails

TreeView mockup review gallery

Static review hub for comparing the current TreeView mockup set without running a Rails app. Each card keeps the existing mockup as the source of truth and adds just enough framing to compare baseline structure, state cues, responsibility boundaries, and focused visual references in one session.

Back to mockup READMEOpen docs index

How to use this gallery

  • Start here when a review needs quick side-by-side comparison across the baseline and focused mockup surfaces.
  • Open the linked full mockup when you want the complete page context or longer notes for one state family.
  • Treat final business copy, routes, permissions, and actions as host-app responsibilities. This gallery stays product-neutral on purpose.
