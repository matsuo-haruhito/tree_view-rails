Responsibility boundary

TreeView-owned cues

Hierarchy depth, expanded or collapsed state, current-row cue, and generated branch lines stay reusable across node types.

Host-owned labels

Node type names, icons, status labels, row actions, and policy wording belong to the host app.

GraphAdapter scope

The adapter can connect heterogeneous records, but this mockup does not define graph traversal or data model behavior.
