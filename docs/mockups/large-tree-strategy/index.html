tree_view-rails

Large-tree strategy decision mock

Static visual reference for comparing how large-tree strategies reduce rendered HTML, child fetching, or host-app work. It intentionally stops at the responsibility boundary: TreeView-owned row output is shown separately from host-app-owned query, route, virtual-scroll, and persistence policy.
