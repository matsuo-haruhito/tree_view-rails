tree_view-rails

Default TreeView rendering mock

Rails を起動せずに、gem 標準 partial が出力する代表的な table / row / cell / data 属性の形を確認するための静的HTMLです。 実アプリ上の playground は demo repo、本体 repo のこの mock は DOM 構造と CSS 適用状態の確認に使います。

Standard table structure

select tree name owner status actions
hide root folder L0 3 Product Platform Team A active
hide child folder L1 selected Admin UI Team B active
leaf file L2 Settings page Team B ready
show child 12 archive L1 archived Legacy reports Team C archived

What this mock covers

  • root / child / leaf の代表行
  • expanded / collapsed / hidden count 表示
  • selected checkbox と disabled checkbox
  • badge / marker / depth label の配置
  • row data 属性、tree state data 属性、row actions partial の列位置
