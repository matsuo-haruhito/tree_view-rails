TreeView breadcrumb path mock

Static HTML for reviewing representative breadcrumb-path output beside the tree's own current-row cue. The path stays outside the table on purpose so reviewers can compare navigation context against row-level state without bringing in routes, Turbo navigation, or host-app wording.

What this mock separates

Surface Shows Keeps outside scope
Breadcrumb path The lineage to the current item, including shallow and deep representative paths. Real route helpers, final labels, authorization-aware visibility, and Turbo navigation.
Current tree row The row that still carries tree-local hierarchy, expansion, and aria-current cues. Cross-page navigation context or host-app breadcrumbs outside the tree.
Combined read How a host app can show both reusable path context and row-local hierarchy cues without conflating their roles. Business-specific navigation wording, responsive overflow policy, and URL design.
