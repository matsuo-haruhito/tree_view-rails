Boundary responsibilities

Use this mock when resource-table review intersects with empty rows, selection checkboxes, row actions, and table-owned visible columns.

TreeView owns the wrapper hook

The reusable empty row exposes data-tree-view-empty-state="true" and the message slot.

Resource tables own exact width

Selection columns, actions, visible columns, and exact colspan remain table-layer decisions.

Host apps own final recovery

Empty copy, permission messaging, reset links, and create actions stay outside the gem contract.
