What this mock covers

  • Representative checkbox visibility for :all, :roots, :leaves, and :none
  • Disabled checkbox rendering and disabled reason placement
  • Mixed parent state as a visual cue rather than a business action workflow
  • Clear boundary between reusable TreeView output and host-app-owned submit behavior
