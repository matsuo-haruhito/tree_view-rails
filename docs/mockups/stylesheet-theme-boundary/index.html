Stylesheet theme boundary

Static reference for comparing TreeView baseline state cues on light, dark, and high-contrast-like host themes. The page keeps runtime stylesheet behavior, public hooks, and CSS custom property policy out of scope.

TreeView-owned cueRows expose reusable current, selected, collapsed, loading, error, and drop-target cues.
Host-owned themeTypography, palette, density, brand tone, and final contrast policy stay with the host app.
