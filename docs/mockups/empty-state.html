Difference from a populated row

tree name owner status actions
hide root folder Product Platform Team A active

Empty rows consume the full content width instead of rendering branch controls.
  • Populated rows keep branch controls, badges, selection, and row actions.
  • Empty rows collapse those row-level controls into one full-width message slot.
  • data-tree-view-empty-state="true" and the empty-row wrapper classes are the reusable surface.
