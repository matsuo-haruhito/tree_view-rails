Focused direction-aware reference

Direction-aware current-row and hierarchy cues

This static mockup compares TreeView-owned baseline hierarchy cues with host-app-owned RTL and writing-direction overrides. It is a review aid only: it does not promote directional selectors, connector pseudo-elements, or spacing classes into public styling hooks.

Responsibility boundary

  • TreeView owns readable baseline current-row, toggle spacing, depth, and hierarchy connector cues.
  • Host apps own final RTL, vertical writing, locale-specific spacing, design-system colors, and business row states.
  • Public styling hook promotion remains outside this page; use the direction-aware styling docs for the current decision.
